# noemir-rpg
NOEMIR(Network Operative Engine Midnight Interactive Rift)(el Rift representa el mundo corrupto por la falta del verdadero prota y un mundo que tuvo que adaptarse)  
El proyecto fue creado para preservar una línea temporal estable y próspera. Tras la muerte del recipiente original, su propósito se corrompió. NOEMIR continúa operando, pero ya no busca la felicidad de la humanidad: busca corregir la anomalía a cualquier costo.
RPG literario de texto basado en el universo de Solo Leveling. El Sistema te eligió como recipiente de contingencia tras el fallo del primer candidato. Ahora tenés que sobrevivir.
Elegís clase, historia de origen y personalizás tu cazador. La narrativa es generada por IA en tiempo real — cada decisión tiene consecuencias reales. Sin inflación de poder. Sin victorias gratuitas. El Sistema no te alienta. Solo registra.
